Diffusion Schrödinger Bridge

0 Abstract

逐步加入高斯噪声可将复杂的数据分布转换为近似高斯分布，逆转上述过程即可得到一个生成模型。

当正向加噪过程由 SDE 给出时，Song[1] 展示了如何通过 score matching 来估计相关反向时间 SDE 的时变漂移项。这种方法的一个局限性是，正向时间 SDE 必须运行足够长的时间，才能使最终分布近似为高斯分布，同时还要确保相应的时间离散化误差 (time-discretization error) 得到控制。

相比之下，求解薛定谔桥（Schrödinger Bridge, SB）问题（即路径空间上的熵正则化最优传输问题），能得到在有限时间内从数据分布生成样本的扩散过程。本文提出了扩散薛定谔桥（Diffusion Schrödinger Bridge, DSB），它是对用于求解 SB 问题的迭代比例拟合（Iterative Proportional Fitting, IPF）过程的一种原创近似，并提供了理论分析与生成建模实验。

第一次 DSB 迭代恢复了(?) Song[1] 提出的方法，且具备使用更短时间间隔的灵活性——因为后续的 DSB 迭代会减小正向（反向）SDE 的终时边缘分布与高斯先验（数据）分布之间的差异。除生成建模之外，DSB 还提供了一个计算最优传输工具，可作为广泛使用的 Sinkhorn 算法在连续状态空间中的类似物。

1 Introduction

基于分数的生成建模（Score-Based Generative Modeling, SGM）是一种近年来发展起来的概率生成建模方法，在多个音频和图像合成任务上展现出最先进的性能。

现有的 SGMs 通常由两部分组成。

首先，将噪声逐步添加到数据中，以获得近似于易采样先验分布（例如高斯分布）的受扰数据分布。其次，使用神经网络来学习反向时间去噪动力学，当从该先验分布初始化时，这种动力学定义了一个生成模型。Song[1] 表明，可以有效地将加噪过程视为一个 SDE，该方程将初始数据分布逐步扰动为近似高斯分布。

相应的反向时间 SDE 是一种非齐次扩散，其漂移项依赖于受扰数据分布的对数梯度，即分数（scores）。在实践中，这些分数使用神经网络和 score matching 技术来近似，而数值 SDE 积分器则用于采样过程。

尽管 SGM 能产生最先进的结果，但样本生成在计算上成本很高。为了从先验（即生成模型）中学习反向时间 SDE，正向加噪 SDE 必须运行足够长的时间以收敛到先验，且步长必须足够小以获得该 SDE 的良好数值近似。通过将生成建模重新表述为薛定谔桥（Schrödinger Bridge, SB）问题，我们缓解了这一问题，并提出了一种求解 SB 问题的新算法。本文具体贡献如下：

Generative modeling as a Schrödinger bridge problem

SB 问题是由 Schrödinger (1932) 提出的著名熵正则化最优传输（Optimal Transport, OT）问题；给定一个具有有限时间范围 TTT 的参考扩散、一个数据分布和一个先验分布，求解 SB 相当于找到与参考扩散最接近的扩散（在路径空间上的 Kullback–Leibler 散度意义下），该扩散在时间 t=0t = 0t=0 时以数据分布为边缘分布，在时间 t=Tt = Tt=T 时以先验分布为边缘分布。

求解该 SB 问题的反向时间扩散提供了一种新的 SGM 算法，与原始 SGM 方法相比，该算法能够使用更短的时间间隔从数据分布近似生成样本。我们的方法与（Genevay 等人, 2018）中的熵正则化 OT 表述不同，后者处理离散分布并依赖于 SB 的静态表述，而我们的方法是针对在路径空间上操作的连续分布的动态方法。它也与（Finlay 等人, 2020）不同，后者通过使用 SB 对偶表述的势来计算漂移项的扩散来近似 SB 解。最后，Wang 等人 (2021) 最近提出通过求解两个而非一个 SB 问题来进行生成建模。与我们相反，他们没有将生成建模表述为计算数据分布和先验分布之间的 SB。

Solving the Schrödinger bridge problem using score-based diffusions

SB 问题可以使用迭代比例拟合（Iterative Proportional Fitting, IPF）来求解（Fortet, 1940；Kullback, 1968；Chen 等人, 2021b）。我们提出了扩散薛定谔桥（Diffusion SB, DSB），这是一种使用基于分数的扩散技术的 IPF 新实现。DSB 不需要对状态空间进行离散化，不需要使用回归来近似势函数，也不需要进行核密度估计。

第一次 DSB 迭代恢复了 Song 等人 (2021) 提出的方法，且具有使用更短时间间隔的灵活性，因为额外的 DSB 迭代会减小正向（相应地，反向）SDE 的终时边缘分布与先验（相应地，数据）分布之间的差异；参见图 1 以作说明。Vargas 等人 (2021) 同时且独立地提出了一种类似于 DSB 的算法；与我们的算法的主要区别在于，他们使用高斯过程来估计 SDE 的漂移项，而我们使用神经网络和分数匹配的思想。


图1：从二维数据分布初始化的参考前向扩散，在扩散时间T = 0.2（共20个离散时间步）时，无法收敛到高斯先验；而从高斯先验初始化的反向扩散，也无法收敛到数据分布。不过，经过5次DSB迭代后，收敛确实会发生。

Theoretical results

我们为 Song 等人 (2021) 的方法提供了首个定量收敛结果。具体而言，我们表明，尽管我们在潜在的极高维空间中模拟朗之万（Langevin）型扩散，但 SGM 方法不会受到混合时间差的影响。此外，我们推导出了连续状态空间中 IPF 的新定量收敛结果，这些结果不依赖于经典的紧性假设（Chen 等人, 2016；Ruschendorf 等人, 1995），并改进了 Léger (2020) 最近的结果。最后，我们表明 DSB 可被视为基于正向/反向扩散的、路径空间上 IPF 动态版本的时间离散化。

Experiments

我们通过生成 MNIST 和 CelebA 等图像数据集来验证我们的方法。具体而言，我们表明使用 DSB 的多个步骤总能改进生成模型。我们还展示了如何使用 DSB 在两个数据分布之间进行插值。
