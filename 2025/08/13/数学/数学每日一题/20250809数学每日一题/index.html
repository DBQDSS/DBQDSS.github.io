题目

求证：lim⁡n→∞nn!n=e\lim_{n\to\infty} \frac{n}{\sqrt[n]{n!}} = elimn→∞​nn!​n​=e

证明

法一（硬放缩）

符号说明：用 log⁡\loglog 来记 ln⁡\lnln

实际上只需证：lim⁡n→∞log⁡nn!n=1\lim_{n\to\infty} \log \frac{n}{\sqrt[n]{n!}} = 1limn→∞​lognn!​n​=1

左侧有，

log⁡nn!n=log⁡n−1nlog⁡(n!)=1n(nlog⁡n−∑i=1nlog⁡i)\log \frac{n}{\sqrt[n]{n!}} = \log n - \frac{1}{n} \log (n!) = \frac{1}{n} \left( n \log n - \sum_{i=1}^n \log i \right) lognn!​n​=logn−n1​log(n!)=n1​(nlogn−i=1∑n​logi)

对于求和项，我们用积分加以逼近。

一方面，

∑i=1nlog⁡i=log⁡2+log⁡3+⋯+log⁡n≤∫2n+1log⁡x dx=xlog⁡x−x∣2n+1=(n+1)log⁡(n+1)−(n−1)\begin{align*} \sum_{i=1}^n \log i &= \log 2 + \log 3 + \cdots + \log n \\ &\leq \int_2^{n+1} \log x \, dx = \bigl. x \log x - x \bigr|_{2}^{n+1} \\ &= (n+1) \log (n+1) - (n-1) \end{align*} i=1∑n​logi​=log2+log3+⋯+logn≤∫2n+1​logxdx=xlogx−x​2n+1​=(n+1)log(n+1)−(n−1)​

另一方面，

∑i=1nlog⁡i=log⁡2+log⁡3+⋯+log⁡n≥∫1nlog⁡x dx=xlog⁡x−x∣1n=nlog⁡n−(n−1)\begin{align*} \sum_{i=1}^n \log i &= \log 2 + \log 3 + \cdots + \log n \\ &\geq \int_1^{n} \log x \, dx = \bigl. x \log x - x \bigr|_{1}^{n} \\ &= n \log n - (n-1) \end{align*} i=1∑n​logi​=log2+log3+⋯+logn≥∫1n​logxdx=xlogx−x​1n​=nlogn−(n−1)​

故，夹逼之，

nlog⁡n−(n+1)log⁡(n+1)+(n−1)n≤log⁡nn!n≤n−1n\frac{n \log n - (n+1) \log (n+1) + (n-1)}{n} \leq \log \frac{n}{\sqrt[n]{n!}} \leq \frac{n-1}{n} nnlogn−(n+1)log(n+1)+(n−1)​≤lognn!​n​≤nn−1​

即，

log⁡(nn+1)+(n−1)n−log⁡(n+1)n≤log⁡nn!n≤n−1n\log (\frac{n}{n+1}) + \frac{(n-1)}{n} - \frac{\log (n+1)}{n} \leq \log \frac{n}{\sqrt[n]{n!}} \leq \frac{n-1}{n} log(n+1n​)+n(n−1)​−nlog(n+1)​≤lognn!​n​≤nn−1​

取 n→∞n \to \inftyn→∞，得

lim⁡n→∞log⁡nn!n=1\lim_{n\to\infty} \log \frac{n}{\sqrt[n]{n!}} = 1 n→∞lim​lognn!​n​=1

□

法二（构造数列）

构造数列 {an}\{a_n\}{an​}，满足：

an=(1+1n)n=(n+1)nnna_n = \left( 1 + \frac{1}{n} \right)^{n} = \frac{(n+1)^n}{n^n} an​=(1+n1​)n=nn(n+1)n​

于是，

lim⁡n→∞an=e\lim_{n\to\infty} a_n = e n→∞lim​an​=e

故，

lim⁡n→∞nn!n=lim⁡n→∞nnn!n=lim⁡n→∞21⋅3222⋅4333⋯(n+1)nnn⋅nn(n+1)nn=lim⁡n→∞(a1⋅a2⋅a3⋯ann⋅nn+1)=e\begin{align*} \lim_{n\to\infty} \frac{n}{\sqrt[n]{n!}} &= \lim_{n\to\infty} \sqrt[n]{\frac{n^n}{n!}} = \lim_{n\to\infty} \sqrt[n]{\frac{2}{1} \cdot \frac{3^2}{2^2} \cdot \frac{4^3}{3^3} \cdots \frac{(n+1)^n}{n^n} \cdot \frac{n^n}{(n+1)^n}} \\ &= \lim_{n\to\infty} \left( \sqrt[n]{a_1 \cdot a_2 \cdot a_3 \cdots a_n} \cdot \frac{n}{n+1} \right) \\ &= e \end{align*} n→∞lim​nn!​n​​=n→∞lim​nn!nn​​=n→∞lim​n12​⋅2232​⋅3343​⋯nn(n+1)n​⋅(n+1)nnn​​=n→∞lim​(na1​⋅a2​⋅a3​⋯an​​⋅n+1n​)=e​
